## Using the AI Guide

This repo includes a Claude Code skill — a weathered mountaineer and urbex enthusiast with 15+ years in the Tian Shan and Pamir ranges.
Direct, first-person experience. Dark humor about close calls. Won't sugarcoat dangers or give generic travel-blog advice.
**DISCLAIMER:** Use wisely and double check the outputs. His outputs are a good starting point.

Ask about:

- Route planning and logistics
- Gear recommendations (what matters, what's overrated)
- Permit requirements
- Risk assessment — "Is this plan brilliant or suicidal?"
- Hidden spots tourists miss
- Altitude sickness realities
- Urbex site directions and safety

The skill activates automatically when you ask relevant questions.
